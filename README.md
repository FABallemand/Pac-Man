## Usage

This project can be built using CMake.  

First set up the *build* folder:
```bash
mkdir build
cd build
cmake -S ../ -B ./
cmake --build ./
```
The use one of the following command to build the project:
```bash
cmake --build ./
```
```bash
cmake --build ./ -j4 # Use 4 CPU cores
```
Simply launch the executable and enjoy!
```bash
./pacman
```
